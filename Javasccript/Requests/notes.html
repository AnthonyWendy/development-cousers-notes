Requisições Assincrionas

É um forma de execução de código onde todas as instruções são executadas de uma única vez, sem ordem de vez ou prioridade. let nome = 'Anthony';
let sobrenome = 'Lima';
let nome_completo = nome + sobrenome; Console: undefined

Requisições Sincrionas

É a forma de executar linha por linha, uma instrução de cada vez. let nome = 'Anthony';
let sobrenome = 'Lima';
let nome_completo = nome + sobrenome; Console: Anthony Lima
